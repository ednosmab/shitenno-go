# State Machine

## Objetivo

Garantir consistência entre todos os comandos.

---

## Regras

Uma única máquina de estados.

Nenhuma transição implícita.

Todos os estados documentados.

Todos os eventos registrados.

---

## Testes

Cada transição deve possuir:

teste válido

teste inválido

teste de regressão
